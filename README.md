# OncoPurpose

[![License: All Rights Reserved](https://img.shields.io/badge/license-All%20Rights%20Reserved-lightgrey)](LICENSE)

OncoPurpose is a research and analysis tool for cancer therapeutics and drug discovery. This repository contains the frontend site and a FastAPI-based backend for analysis APIs and integrations with public data sources (ChEMBL, PubChem, DailyMed).
